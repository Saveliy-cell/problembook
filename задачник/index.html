Задачи с целыми числами

  • Решите данное математическое выражение, с помощью функции print():
    (8432 + 7812) : 4 * 32
  • Дана масса M в килограммах. Используя операцию деления нацело, найти количество полных тонн в ней.
  • Дан размер файла в байтах. Используя операцию деления нацело, найти количество полных килобайтов, которые занимает данный файл (1 килобайт = 1024 байта).
  • Дано трехзначное число x. Найти сумму его цифр.
  • Дан номер некоторого года. Определить соответствующий ему номер столетия, к примеру, 1805 - 19 столетие.
  • Дано число x. Первую цифру этого числа переносят на последнее место. Вывести полученное число
  • Дано трехзначное число x. Вывести число, полученное при перестановке разряда сотен и десятков исходного числа (например, из числа 123 получится 213).
  • Дано целое число, большее 999. Используя одну операцию деления нацело и одну операцию взятия остатка от деления, найти цифру, соответствующую разряду сотен в записи этого числа.
  • Дни недели пронумерованы следующим образом: 1 — понедельник, 2 — вторник. Дано целое число x, лежащее в диапазоне 1–365. Определить номер дня недели для K-го дня года, если известно, что в этом году 1 января было вторником.
  • С начала суток прошло xb> секунд (N — целое). Найти количество полных минут, прошедших с начала суток.

Задачи с условным оператором

  • Дано целое число A. Проверить истинность высказывания: «Число A является нечетным».
  • Даны два целых числа: A, B. Проверить истинность высказывания: «Является ли одно из чисел A или B нечетным».
  • Даны два числа: A, B. Вывести большее из них.
  • Дано целое число, лежащее в диапазоне 99–999. Вывести его строку описание вида «четное двузначное число», «нечетное трехзначное число».
  • Дано целое положительное число. Проверить истинность высказывания: «Данное число является четным двузначным»
  • Дано целое число в диапазоне 1–7. Вывести строку — название дня недели, соответствующее данному числу (1 — «понедельник» и т.д.).
  • Дан номер месяца — целое число в диапазоне 1–12 (1 — январь, 2 — февраль и т.д.). Вывести название соответствующего времени года («зима», «весна», «лето», «осень»)
  • Дан номер месяца — целое число в диапазоне 1–12 (1 — январь, 2 — февраль и т.д.). Определить количество дней в этом месяце для не високосного года.
  • Пользователь вводит число, указывающие, то насколько верно выполнена работа. Определить оценку, которую получил пользователь. Оценка выставляется по следующим критериям:
    1. 89% и больше – 5
    2. 69% - 88% – 4
    3. 49% - 68% – 3
    4. 48% и меньше – 2
  • Пользователь должен ввести пароль. При этом требования к паролю следующие:
    1. Длина пароля должна быть больше 8 символов.
    2. Пароль должен содержать хотя бы 1 заглавную букву.
    3. Пароль должен содержать цифры.

Задачи с циклами

  • Даны целые числа K и N (N > 0). Вывести N раз число K (решить с помощью цикла While).
  • Дано целое число N (>0). Если оно является степенью числа 3, то вывести TRUE, если не является — вывести FALSE (решить с помощью цикла While).
  • Дано целое число N (>0). Используя операции деления нацело и взятия остатка от деления, вывести все его цифры, начиная с самой правой (решить с помощью цикла While).
  • Дано целое число N (>0). Используя операции деления нацело и взятия остатка от деления, вывести сумму всеx его цифр (решить с помощью цикла While).
  • Даны два целых числа A и B (A < B). Найти сумму квадратов всех целых чисел от A до B включительно (решить с помощью цикла While).
  • Необходимо вывести на экран числа от 1 до 5 (решить с помощью цикла For).
  • Дано целое число N (>0). Необходимо вывести все четные числа от 0 до N (решить с помощью цикла For).
  • Дано вещественное число A и целое число N (>0). Найти A в степени N (решить с помощью цикла For).
  • Дан список автомобилей:

    cars = ["BMW", "Mercedes", "Volvo", "Toyota", "Tesla"]

    Необходимо вывести каждый элемент списка cars, (решить с помощью цикла For).
  • Дан словарь:

    client = {
    "name" : "Alex",
    "phone_number" : "12345",
    "age" : "25"
    }

    Необходимо вывести каждый элемент словаря в формате: ключ : значение, (решить с помощью цикла For).

Задачи с модулем random

  • Моделирование бросков игрального кубика c 6 гранями. Программа выводит результаты бросков — выпавшее число, которое написано на грани кубика.
  • Выведите случайный элемент из списка ['apple', 'banana', 'cherry', 'date', 'elderberry'].
  • Сгенерируйте случайное число с плавающей точкой в диапазоне от 0 до 1 и определите, попадает ли оно в интервал от 0,2 до 0,8.
  • Сгенерируйте IP-адрес. IP-адрес состоит из четырёх чисел из диапазона от 0 до 255, разделённых точкой.
  • Напишите программу, которая будет генерировать пароль, состоящий из 5 цифр.

Задачи с модулем math

  • Вычислите модуль числа -7 и выведите результат на экран.
  • Вычислите квадратный корень из числа 25 и выведите результат на экран.
  • Найдите значение синуса угла в 45 градусов и выведите его на экран.
  • Дано число r. Вычислите площадь круга, используя формулу площади круга S=πr2 (π = 3,14). Выведите результат с точностью до двух знаков после запятой.
  • Найдите значение выражения log10(x2 +y2) для заданных чисел x и y.
